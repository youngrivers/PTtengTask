平台模式 常见问题 学习贴士

Q : 是否需要英语基础？

A : 写代码必须要有英语基础，否则遇到问题没法去找英文资料。但是对英语的要求不太高，用好各种词典，能看懂技术答疑就好了。 所以不要怵，认真学，少用各种汉化的编辑器，多看英文文档。

Q : 为什么没有提供视频教学和教程？

A : 原因一：学习IT技术，经常会遇到问题。遇到问题是写代码的常态，一旦遇到问题，就需要学会自己去百度，去主动解决问题。这是工程师必备的能力，所以，在基础的知识点上，你看视频看教程能学会一点，但是以后遇到没视频没教程的怎么办？这种能力怎么培养？要从一开始就养成好习惯。 原因二：代码是写出来的，不是看视频看教程看出来的。代码调错，不同方法的尝试，遇到的问题怎么解决，尝试不同的方式，这些都是教程所提不到的，每遇到一个问题，都是一个学习和检验自己的机会。程序员的一生就是和Bug纠缠不清的一生。

Q : 我应该学什么职业？

A : 可在职业殿堂上写的是比较清楚，基本上分为有基础，没基础。能力高，能力低。选择一些入门比较简单快捷的职业是没问题的，当时记着，如果你想在这个领域走的远，就必须是把所有的基础课程补上。

Q : 什么是首席大师兄？

A : 首席大师兄是每一个班里乐意帮助别人，进度不错的人。首席大师兄可以直接向内门弟子请教。所以，对于其他的普通同学来讲，你们有问题，就需要去问自己的首席大师兄和上期班级的首席大师兄。

Q : 线下怎么收费，线下怎么学习，时间是多长？

A : 线下目前的收费方式是每个职业都有不同。从一个月400块钱开始，每期都会增长400，直到某个上限为止。线下的学习方式跟线上基本相同，只是线下的氛围更好，要求更严格，遇到问题可以随时有人指导，速度快的话还可以有真实的项目实践，表现的好，会有工作机会推荐。同时，会有很多在线上无法学到的内容，比如说职业素养，时间观念，做事心态，这些都是潜移转化出来的。 线下一个月算一期，报名要去论坛，提前付押金，押金不退。因地方有限，所以每期都有名额限制。

Q : 初级工程师怎么定义？他能达到什么样的能力?

A : 修真院对于初级工程师的定义非常严格。能够自己独立完成一个项目的，在修真院眼里才是一个初级工程师。所以，修真院说到的初级工程师，相当于原始方式半年到一年的甚至是两年工作经验的人。 初级工程师应该学会项目管理，进度控制，版本发布，Bug修复流程，代码规范，需求分析，性能测试等等。

Q : 我是否应该去参考师兄的代码？

A : 绝对不要刚开始就去参考师兄的代码，想想你们的师兄在没有可以参考的情况下是怎么做出来的。这种思考比你把任务做出来更重要，因此，你需要的是拿到任务之后，先去自己想一想怎么解决，然后自己去百度，当遇到问题，或者是需要跟其他人对比一下思路的时候，再去看师兄的代码，这样才是一个标准的学习思路。

Q : 是否完全适用于0基础？

A : 适用于0基础，但是不适用于没有自学能力的0基础。如果你没有自学的能力，总是埋怨他人，显然你不适合这里。这里聚集的是愿意去学，主动去学，有恒心有毅力有志向唯独没有方向和方法的人。程序员最重要的能力就是学习能力了，而这种能力是需要不断的去强化和训练的，所以这里从一开始就避免去从视频和教程入手，而是希望你们能够通过自己的思考完成任务。

Q : 我的时间比较零散怎么办？

A : 修真院的任务模式，对时间并无特殊要求。只是有大段的时间，会对你的学习更有帮助。如果时间比较零散，也没有关系，每个任务之间都是相对独立的。你完成了一个任务，就是学会了任务对应的技能。 所以，如果你每天能够抽出3个小时的时间，也是可以坚持做下去的，但是时间会比较长。
